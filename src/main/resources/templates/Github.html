Learning GitHub

Fredrico Corleone

So far in our java bootcamp we learned how to transfer code from a program called intellij to a web data base called Github using a series of commands in a terminal that created a conduit from git hub to intellij and vice versa. We also learned how to “commit” and “push” this lets us the user make updates to these coding projects online called repositories. These repositories make your code available to all kinds of people in the internet.
